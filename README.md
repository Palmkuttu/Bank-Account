# Assignment 5: Bank Account

This project implements a simple bank account system using Python OOP.

## Files

- banking.py  
  Contains the Transaction and Account classes.

- test_banking.py  
  Contains pytest tests for the banking classes.

## Features

- Create transactions with timestamps
- Deposit money
- Withdraw money
- Calculate account balance

## Run Tests

Install pytest:

pip install pytest

Run tests:

pytest
