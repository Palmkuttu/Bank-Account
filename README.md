# Bank Account (OOP Python)

## 📌 Description

This project implements a Bank Account system using Object-Oriented Programming in Python.

The program allows users to:

* Create a bank account
* Deposit money
* Withdraw money
* Check account balance

## 🧱 Files Included

* `banking.py` → Main Account class implementation
* `test_banking.py` → Test cases for the account
* `.gitignore` → Ignored files

## ▶️ How to Run

1. Open the project in PyCharm or terminal
2. Run the test file:

   ```
   python test_banking.py
   ```

## 🧠 Concepts Used

* Classes and Objects
* Methods
* Encapsulation
* Special methods (`__str__`, `__repr__`)
